Hello, we are Qaclana

We are glad to meet you.

Qaclana is an open source project, aimed to creating a modern Web Application Firewall in the open. We believe that security is something that should be easy to adopt, be it by big corporations with a dedicated security team, be it by one-person web shops.


At this stage, there's no company behind Qaclana and it's being financially supported by the project's founder.
